How Clifixer Works

Get started in 3 simple steps. No IDE lock-in, just pure debugging power.

Install the Clifixer SDK globally to access it anywhere:

bash
$ npm install -g clifixer
# Installs CLI globally

Configure with your API token (get it from your dashboard):

bash
$ npx clifixer init --token=YOUR_TOKEN
# Creates config file at ~/.clifixer

Start debugging with optional context (logs, specific files):

bash
$ npx clifixer chat --context=./lib
> [Clifixer] Analyzing 12 files in ./lib...
> Ask me about bugs (type 'exit' to quit):
> Why is my API returning 500 errors?
> [Clifixer] Found unhandled exception in routes.js:42...
